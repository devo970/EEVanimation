# HVAC Training Tools — Fujitsu VRF & Ductless

Interactive single-file HTML training visuals for Fujitsu ductless 
and VRF system diagnostics and instruction.

Built for use in PowerPoint presentations via the Web Viewer add-in,
or opened directly in any browser.

---

## How to Use in PowerPoint

1. Open PowerPoint
2. Insert → Get Add-ins → search **Web Viewer** → Add
3. Insert → My Add-ins → Web Viewer
4. Paste the link from the table above
5. Resize the frame to fill the slide
6. Internet connection required during presentation

---

## How to Use Standalone

Download any HTML file and open it in Chrome or Edge.
No internet required after download. No installation.

---

## Adding New Tools

Upload the HTML file to this repo, then add a row to the table above.
All PowerPoint slides using Web Viewer update automatically.
